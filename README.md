## Descrição 📄

O Memory Game Template é um aplicativo Android desenvolvido em Kotlin que permite aos jogadores criar suas próprias coleções de cartas e compartilhá-las com outros usuários. O jogo é uma versão personalizável do clássico jogo da memória, onde os jogadores têm que encontrar pares de cartas correspondentes.

## Demonstração 🎥

Confira a [demonstração em GIF do Memory Game Template](https://raw.githubusercontent.com/wagarcdev/MemoryGameTemplate/master/app/src/main/res/blob/game_gif.gif) para ver o jogo em ação!

## Recursos Principais ⭐️

- 🎴 Customização de Cartas: Os jogadores podem criar suas próprias coleções de cartas personalizadas para jogar o jogo da memória. Cada carta pode conter imagens, texto ou qualquer outro conteúdo desejado.

- 🔥 Integração com o Firebase: O aplicativo utiliza as bibliotecas do Firebase, incluindo o Firebase Firestore e o Firebase Storage, para permitir que os jogadores compartilhem suas coleções de cartas com outros usuários.

- 📊 Vários Níveis de Dificuldade: O jogo oferece três opções de níveis de dificuldade com base no número de cartas: 8, 18 e 24. Os jogadores podem escolher o nível que desejam jogar.

- 🎉 Efeitos Visuais: O aplicativo utiliza a biblioteca Confetti para adicionar efeitos visuais divertidos quando os jogadores encontram todos os pares de cartas.

## Uso 🚀

O aplicativo é desenvolvido em Kotlin e utiliza as bibliotecas do Firebase, como o Firebase Firestore e o Firebase Storage, para a integração com a plataforma. Certifique-se de configurar as chaves de API corretamente no projeto.
